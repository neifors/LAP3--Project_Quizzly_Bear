QUIZZLY BEARS API

The API is a mechanism which allows queries in the semantic database. It provides access to information from the data catalogue and URIs. These queries can be carried out based on a series of criteria selected by the user and using uniform resource identifiers (URI).

RESPONSE FORMAT

JSON is the default response format

Available References

Data catalogue

Set of calls which allow the API catalogue queries.

Operation Description Method Example
