Quizzly Bears API

The API is a mechanism which allows queries in the semantic database. It provides access to information from the data catalogue and URIs. These queries can be carried out based on a series of criteria selected by the user and using uniform resource identifiers (URI).

RESPONSE FORMAT

JSON is the default response format

Available References

Data catalogue

Set of calls which allow the API catalogue queries.

Operation Description Method Request Body Expected Example
To obtain all of the users information /users GET N/A https://quizzlybears.azurewebsites.net/users
To obtain data set with a specific username /users/username/:username GET N/A https://quizzlybears.azurewebsites.net/users/username/futureproof
To obtain data set with a specific id /users/id/:id GET N/A https://quizzlybears.azurewebsites.net/users/id/625a90bb35a40d43d86054b4
To create a new user /users/register POST { username: "username", password: "password"} https://quizzlybears.azurewebsites.net/users/register
To login /users/login POST { username: "username", password: "password"} https://quizzlybears.azurewebsites.net/users/login
